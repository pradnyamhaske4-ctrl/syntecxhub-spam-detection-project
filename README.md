# syntecxhub-spam-detection-project
This project detects whether a message is SPAM or HAM using Machine Learning.

🧠 Technologies Used

- Python
- Scikit-learn
- TF-IDF Vectorizer
- Naive Bayes

⚙️ Features

- Text preprocessing
- Model training
- Accuracy & evaluation
- Real-time message prediction

▶️ How to Run

1. Install libraries:
   pip install pandas scikit-learn joblib

2. Run:
   python spam_detection.py

📊 Output

- Accuracy
- Confusion Matrix
- Spam Prediction

📁 Files

- spam_detection.py
- spam.csv
- spam_model.pkl
